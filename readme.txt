
🛍️ API de Gerenciamento de Itens com FastAPI
Python
FastAPI

Uma API RESTful para gerenciamento de itens com operações CRUD e transformações de preços, implementando conceitos de programação funcional.

📦 Pré-requisitos
Python 3.10+
Pip
Virtualenv (recomendado)

🚀 Instalação
Clone o repositório:

git clone https://github.com/AndredsLima/ProgramacaoFuncional.git

Crie e ative um ambiente virtual:

python -m venv venv
source venv/bin/activate  # Linux/Mac
.\venv\Scripts\activate  # Windows


Instale as dependências:
📋 Arquivo requirements.txt contém todas as dependências necessárias:

pip install -r requirements.txt

🏃 Executando a Aplicação

uvicorn main:app --reload
